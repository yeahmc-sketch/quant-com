# 因子研究补充报告

## 研究背景
基于2小时的全网因子研究，针对现有MF、B2、DTK10策略进行因子补充和优化建议。

## 一、MF策略（10天Fusion20+DMA）补充因子

### 1. 价量关系增强因子
#### 高低位放量因子
- **逻辑**：高位放量预示主力出货，低位放量预示主力进场
- **公式**：`volume_ratio = volume / mean(volume, 20)`
- **应用**：当股价创新高且volume_ratio > 1.5时预警；当股价创新低且volume_ratio > 2.0时关注

#### 量价背离因子
- **逻辑**：价格创新高但成交量萎缩，或价格创新低但成交量放大
- **公式**：`price_volume_divergence = (close - delay(close, 5)) / mean(volume, 20)`
- **应用**：捕捉量价背离信号

### 2. 波动率因子
#### 时变波动率因子
- **逻辑**：自适应市场波动状态
- **公式**：`adaptive_vol = std(close, 20) / mean(close, 20) * 100`
- **应用**：根据波动率调整仓位大小

#### 波动率偏度因子
- **逻辑**：捕捉波动率的非对称性
- **公式**：`vol_skew = (std(close, 20)^3) / mean((close - mean(close, 20))^3, 20)`
- **应用**：识别波动率异常模式

## 二、B2策略（20天Fusion20+2xLGBM+NavDD8%）补充因子

### 1. 中周期因子
#### 20日动量加速度因子
- **逻辑**：捕捉动量变化的加速度
- **公式**：`momentum_accel = (momentum_20 - momentum_10) / momentum_10`
- **应用**：识别动量拐点

#### 相对强度因子
- **逻辑**：个股相对于行业的相对表现
- **公式**：`relative_strength = close / mean(close, 20) - industry_mean`
- **应用**：选择行业领先股

### 2. 技术指标增强因子
#### 多时间框架均线系统
- **逻辑**：融合5日、10日、20日均线信息
- **公式**：`ma_system = (close - ma5) * 0.5 + (close - ma10) * 0.3 + (close - ma20) * 0.2`
- **应用**：提供更全面的趋势信号

#### 成交量加权平均价格因子
- **逻辑**：考虑成交量的价格权重
- **公式**：`vwap_factor = (sum(close * volume, 20) / sum(volume, 20) - close) / close`
- **应用**：识别价格与内在价值的偏离

## 三、DTK10策略补充因子及修正

### 1. 计算错误修正
#### 正确的夏普比率计算
- **问题**：当前使用sqrt(252/5)计算夏普比率
- **修正**：使用sqrt(252)进行年度化
- **公式**：`sharpe_ratio = annual_return / annual_volatility * sqrt(252)`

### 2. 因子库扩展
#### Alpha158精选因子
- **建议添加**：Alpha158中的20个表现最优因子
- **重点考虑**：动量类、波动率类、成交量类因子
